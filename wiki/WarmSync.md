# Синхронизация теплопотерь

Набор инструментов на панели **«Теплопотери»** для переноса значений температуры и теплопотерь между пространствами Revit и Excel. Ниже — полный порядок работы.

## 1. Подготовка проекта

Для работы плагина в проекте должен быть параметр **«ID элемента»**. Добавить его можно двумя способами.

### Способ 1

Скопировать ведомость **«Э_пространства»** из модели «Пространства»:

`Y:\12-BIM\Стандарт\Общие семейства и мануалы ко всем шаблонам\Авто заполнение теплопотерь в пространствах\Пространства.rvt`

<img width="354" alt="Ведомость Э_пространства" src="https://github.com/user-attachments/assets/cdea1d3d-f92c-48d2-a894-1a51e2de8a1b" />

### Способ 2

Перейти во вкладку **«Управление»** и нажать кнопку **«Параметры проекта»**.

<img width="820" alt="Параметры проекта" src="https://github.com/user-attachments/assets/a785d87f-999e-4d83-b698-c66498def30d" />

Нажать **«Добавить…»**, заполнить окно как на скриншоте ниже и нажать **«ОК»**.

<img width="604" alt="Окно добавления параметра" src="https://github.com/user-attachments/assets/2f3ddd34-c31a-4bf6-a137-8cd83e0d7976" />

## 2. Создание пространств

Для создания пространств воспользуйтесь плагином **[Создание пространств](CreateSpaces)** на вкладке ToadTools.

## 3. Заполнение имён пространств

Нажмите **Теплопотери → Заполнение имён пространств**.

<img width="604" alt="Заполнение имён пространств" src="https://github.com/user-attachments/assets/2e1ae224-9dce-4dfb-8bcc-c6b329e4b179" />

## 4. Запись ID пространств

Для записи ID пространств в параметр «ID элемента» нажмите **Теплопотери → Запись ID пространств**.

<img width="576" alt="Запись ID пространств" src="https://github.com/user-attachments/assets/c5710f1b-dc38-4e24-9f47-6ea5a9dba34e" />

## 5. Экспорт пространств в Excel

Нажмите **Теплопотери → Экспорт пространств в Excel**.

<img width="558" alt="Экспорт пространств в Excel" src="https://github.com/user-attachments/assets/637cc8e0-423a-4370-87ad-cb99946aa54f" />

Плагин предложит выбрать путь для сохранения Excel-файла. После этого можно открыть файл и заполнить в нём значения температуры и теплопотерь.

<img width="820" alt="Excel-файл с пространствами" src="https://github.com/user-attachments/assets/6d238333-9b84-4e64-ad48-1ad3ebbb7319" />

## 6. Импорт значений из Excel

Для импорта значений температуры и теплопотерь из Excel обратно в Revit нажмите **Теплопотери → Импорт значений из Excel**.

<img width="573" alt="Импорт значений из Excel" src="https://github.com/user-attachments/assets/be98fdd2-01a1-456f-86fc-b564bd9c6e5b" />

После этого значения загрузятся в пространства и будут отображаться в марках на видах.
