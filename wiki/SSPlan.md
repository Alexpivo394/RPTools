# Структурная схема

Плагин строит структурную (однолинейную) схему по выбранной панели на чертёжном виде.

## 1. Запускаем плагин кнопкой «Структурная схема» на панели плагинов

<img width="811" alt="Кнопка «Структурная схема» на ленте" src="https://github.com/user-attachments/assets/c2130b42-af86-400f-96ff-547737995732" />

> [!WARNING]
> Запускать плагин нужно на чертёжном виде, на котором вы хотите создать схему.

## 2. Заполняем основные поля плагина

<img width="616" alt="Окно плагина" src="https://github.com/user-attachments/assets/bf4b42a7-64ab-43f5-813a-7ac9ab4986d0" />

1. Из выпадающего списка выбираем панель, структурную схему которой хотим создать. В список попадают только панели с подключёнными электрическими цепями. Панели отображаются по **Имени панели**; если этот параметр не заполнен, панель в списке не найти.
2. Этой кнопкой можно выбрать панель на плане. При нажатии окно скрывается, и Revit предлагает выбрать панель.
3. Здесь выбираем семейство аннотации, из которого будем делать схему. Нужно выбирать семейство **RR_CC_Видеокамера**.

> [!NOTE]
> Если семейство камеры не загружено в проект, оно лежит на общем диске:
> `Y:\12-BIM\Стандарт\Общие семейства и мануалы ко всем шаблонам\Семейства ЭОМ, СС\2D Типовые аннотации\RR_CC_Видеокамера.rfa`

4. Смещение по оси **X** между камерами — для стандартной горизонтальной схемы равно `25`.
5. Смещение по оси **Y** между камерами — для стандартной горизонтальной схемы равно `0`.

## 3. Нажимаем «Создать»

На чертёжном виде появляется структурная схема.
