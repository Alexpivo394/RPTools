# Основание отделки

Плагин переносит значение параметра из стены-основания в параметр отделочной стены.

> [!NOTE]
> Перед запуском плагина необходимо выбрать параметр и заполнить его значение: у стен-основ значение этого параметра заполняется согласно материалу стены, а у отделочных стен значение этого же параметра используется для фильтрации отделочных стен и должно быть одинаковым для всех отделочных стен (*отделка, отделка стен и т.д.*).

1. Плагин можно запускать как на текущем виде, так и для всей модели. Открыв нужный вид, запустите плагин **«Основание отделки»** во вкладке **АР**.

<img width="820" alt="Кнопка «Основание отделки» на ленте" src="https://github.com/user-attachments/assets/1e48d0a0-e417-4d3c-95e7-e678151c6528" />

2. В открывшемся окне задайте три значения:

   - имя параметра, значение которого будет передаваться из стены-основы;
   - значение этого параметра, заданное для отделочных стен;
   - имя параметра отделочных стен, в который необходимо записать значение.

   Далее выберите вариант запуска — для элементов на текущем виде или на всей модели — и нажмите **«Запуск»**.
