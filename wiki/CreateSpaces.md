# Создание пространств

<img width="820" alt="Кнопка «Создание пространств» на ленте" src="https://github.com/user-attachments/assets/18c3ae59-8a12-4672-9a8f-cd324a0312d9" />

Плагин создаёт в проекте пространства на основе помещений из связанного файла АР.

## 1. Обзор интерфейса

<img width="810" alt="Окно плагина" src="https://github.com/user-attachments/assets/1ec390b3-4de8-46a6-9dd0-e8c4d24bdc12" />

1. Поле выбора связанного файла, из которого берутся помещения для переноса.
2. Настройка переноса значений параметров из помещений в пространства.
3. Переключатель режима работы: если включён — плагин создаёт пространства; если выключен — только обновляет значения параметров в существующих пространствах.
4. Кнопки экспорта и импорта настроек переноса параметров.
5. Кнопка запуска плагина.

## 2. Работа плагина

После открытия окна выберите связанный файл, из которого будете брать помещения, — появится список параметров для переноса. При необходимости настройте перенос параметров. Далее нажмите **«Запуск»** и дождитесь сообщения об успехе.

> [!NOTE]
> Для корректного создания пространств лучше сразу задать перенос следующих параметров на аналогичные из помещения: **Имя**, **Номер**, **Смещение сверху**.

> [!WARNING]
> Плагин создаёт пространства для всех помещений в связанном файле. Если в нём несколько корпусов, а вам нужны пространства только по одному корпусу, можно настроить перенос параметра с принадлежностью к корпусу, а затем удалить ненужные пространства через ведомость.
