# Открывание дверей

Плагин заполняет параметр **«Открывание»** для всех дверей на текущем виде. Если в проекте нет такого параметра, плагин сам его создаёт при первом запуске.

Для запуска перейдите во вкладку **ToadTools** и нажмите кнопку **«Открывание дверей»**.

<img width="820" alt="Кнопка «Открывание дверей» на ленте" src="https://github.com/user-attachments/assets/2a25865f-9f94-4563-b178-92c0d74a6f64" />

Для правых дверей параметр остаётся пустым, для левых в параметр записывается **«л»**.

> [!NOTE]
> Плагин работает только с семейством **«Дверь»** или его аналогами, открывание которых меняется по развороту через стрелочки.
>
> Для дверей, у которых открывание меняется через параметры экземпляра или типа, плагин не работает.
